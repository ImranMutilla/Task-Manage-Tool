# Smart Task Assistant

Todoist 风格任务组织 + 简约现代视觉的轻量任务管理系统。

## 本轮升级重点

- ✅ Upcoming 重构：按日期分组（周维度）+ 顶部周导航（Prev/Today/Next）+ 每天独立 `+ Add task`
- ✅ Today 执行视图：Overdue / Due Today / No Time 分区 + 今日摘要
- ✅ 日期快捷选择：No date / Today / Tomorrow / This weekend / Next week / Pick a date
- ✅ 新增任务详情面板：点击任务即可查看完整信息并跳转编辑
- ✅ Tag View 可点击导航，标签继续支持统一颜色映射
- ✅ Repeat 轻量版：none / daily / weekday / weekly / monthly，完成后自动生成下一次任务
- ✅ 列表信息层级优化：标题、优先级、项目、时间、标签、描述摘要
- ✅ 空状态与上下文快速添加入口（Inbox / Today / Upcoming）

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```
